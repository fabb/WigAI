L55: ### Developer Context (Guardrails)
L56: 
L57: - This is a **host-required smoke harness**. It validates a *running* Bitwig instance with the WigAI extension enabled. Do not attempt to make this pass in CI.
L58: - Default behavior must be **read-only**. Mutations require an explicit opt-in.
L59: - Output must be **actionable**:
L60:   - Always print the resolved MCP URL and whether safe-mode vs mutation-mode ran.
L61:   - On failure, print which step failed, the MCP method/tool name, and any returned `error.code`/`error.message`.
L62:   - Exit codes: `0` on pass, `1` on any failed assertion/timeout/unreachable server.
L63: - Use the **MCP Java SDK client utilities** if available; do not hand-roll protocol framing unless necessary.
