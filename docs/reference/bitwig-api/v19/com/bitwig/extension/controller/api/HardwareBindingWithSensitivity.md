# HardwareBindingWithSensitivity

- Kind: interface
- Package: `com.bitwig.extension.controller.api`

A HardwareBinding that has some sensitivity setting.

**Extends:** `com.bitwig.extension.controller.api.HardwareBinding`

## Methods

### setSensitivity(double)

Sets the sensitivity of this binding.

**Parameters:**
- `sensitivity` (type double)

**Returns:** type void

### removeBinding()

Removes this binding between its source and target so it is no longer in effect.

**Returns:** type void
