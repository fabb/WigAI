# Groove

- Kind: interface
- Package: `com.bitwig.extension.controller.api`

An interface representing the global groove settings of the project.

## Methods

### getEnabled()

Returns the enabled state of the groove.

**Returns:** type Parameter - an object that provides access to the groove on/off setting

### getShuffleAmount()

Returns the object that represents the shuffle amount in Bitwig Studio.

**Returns:** type Parameter - an ranged value object that provides access to the shuffle amount

### getShuffleRate()

Returns the object that represents the shuffle rate in Bitwig Studio.

**Returns:** type Parameter - an ranged value object that provides access to the shuffle rate

### getAccentAmount()

Returns the object that represents the accent amount in Bitwig Studio.

**Returns:** type Parameter - an ranged value object that provides access to the accent amount

### getAccentRate()

Returns the object that represents the accent rate in Bitwig Studio.

**Returns:** type Parameter - an ranged value object that provides access to the accent rate

### getAccentPhase()

Returns the object that represents the accent phase in Bitwig Studio.

**Returns:** type Parameter - an ranged value object that provides access to the accent phase
