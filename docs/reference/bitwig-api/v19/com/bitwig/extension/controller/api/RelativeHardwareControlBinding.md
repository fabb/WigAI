# RelativeHardwareControlBinding

- Kind: interface
- Package: `com.bitwig.extension.controller.api`

Represents a binding from a RelativeHardwareControl to some target.

**Extends:** `com.bitwig.extension.controller.api.HardwareBindingWithSensitivity`

## Methods

### setSensitivity(double)

Sets the sensitivity of this binding.

**Parameters:**
- `sensitivity` (type double)

**Returns:** type void

### removeBinding()

Removes this binding between its source and target so it is no longer in effect.

**Returns:** type void
